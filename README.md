# Netflix Content Analysis Dashboard

## Project Overview

This Power BI dashboard analyzes Netflix content and provides insights into:

* Total Titles
* Movies vs TV Shows
* Content Released by Year
* Top Countries by Content
* Content Ratings
* Top Genres

## Tools Used

* Power BI
* Microsoft Excel / CSV Dataset

## Dashboard Preview

![Dashboard Screenshot](Screenshot%20\(335\).png)

## Key Insights

* Movies account for the majority of Netflix content.
* The United States has the highest number of titles.
* TV-MA is the most common content rating.
* Content releases increased significantly after 2015.
